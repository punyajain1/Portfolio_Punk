# Vintage Mac Portfolio

A highly detailed vintage Macintosh desktop interface (System 7/Mac OS 9 aesthetic) portfolio built with Next.js, TypeScript, and Tailwind CSS.

## Features

- **Vintage Desktop Interface**: Classic Menu Bar, Desktop Icons, and Windows.
- **Modern Touches**: Translucent Dock with animations.
- **Widgets**: Profile and Status widgets.
- **Loading Animation**: "oh hello" intro.
- **Tech Stack**: Next.js 14, TypeScript, Tailwind CSS, Framer Motion, Lucide React.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser.

## Customization

-   **Icons**: Edit `src/components/desktop/Desktop.tsx` and `src/components/dock/Dock.tsx`.
-   **Content**: Update `src/components/desktop/Desktop.tsx` for window content.
-   **Styles**: `src/app/globals.css` and Tailwind classes.
